# TcpListener Human Machine
- **操作方法**
  - 連上椅墊wifi
    - 設定IPv4位址為10.0.0.2
    - 設定IPv4遮罩為255.255.255.0
    - 設定IPv4匝道為10.0.0.1
    - 設定DNS為10.0.0.1
  - Browse
    - 選擇存檔位置
  - Save File
    - 勾選後開始存檔
  - Sampling Rate
    - 設定椅墊取樣率
  - Zeroing Offsets
    - 波型offsets歸零
  - Connect連接至椅墊
  - Disconnect斷開連接
